== Installation ==

The plugin can be installed like any other plugin.You can search from WordPress Dashboard (Udinra Easy Digital Downloads) and click install to install it.You can also download it and upload it to your server wp-content/plugins directory and extract the zip file.

== Frequently Asked Questions ==

The plugin refreshes shop every day. But in case you want to refresh shop immeadiately after any product related change click on 
Refresh Button on configuration page.
